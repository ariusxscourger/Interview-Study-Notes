## 🚀 FINAL WEEK CHECKLIST

### Technical Readiness
- [ ] Can solve LeetCode Medium in 25 min, Hard in 40 min
- [ ] Can design 5 systems from memory (URL Shortener, Rate Limiter, Chat, Feed, Cache)
- [ ] Can explain 11 coding patterns with code templates
- [ ] Can articulate 3 system design trade-offs per design
- [ ] Can answer "why this company?" with specific details

### Behavioral Readiness
- [ ] 10 STAR stories written, recorded, timed (2-3 min each)
- [ ] Stories cover: Conflict, Leadership, Mistake, Learning, Pressure, Ambiguity, Influence, Growth
- [ ] Stories mapped to target company values
- [ ] Practiced "Tell me about yourself" (90 sec version)

### System Design Readiness
- [ ] Can draw 5 architectures in 5 min each
- [ ] Knows CAP/PACELC, consistent hashing, sharding strategies
- [ ] Can discuss caching strategies (cache-aside, read-through, write-behind)
- [ ] Knows back-of-envelope math for 1M/10M/100M users

### Negotiation Readiness
- [ ] 3 competing applications in pipeline
- [ ] Target numbers: Base, Equity, Sign-on, Refresh, Total
- [ ] Scripts memorized for top 5 objections
- [ ] Walk-away number defined

### Logistics
- [ ] Outfit ready (comfortable, professional)
- [ ] Location: Quiet, good lighting, neutral background
- [ ] Tech: Laptop charged, charger, backup internet (hotspot)
- [ ] Accounts: LeetCode, GitHub, LinkedIn, portfolio updated
- [ ] Documents: Resume (PDF), references, portfolio links ready
- [ ] Timezone confirmed, calendar invites accepted

### Health & Mindset
- [ ] 7-8 hrs sleep for 3 nights before
- [ ] Hydration + nutrition plan
- [ ] Visualization exercise done (see yourself succeeding)
- [ ] Gratitude: Thanked mentors/peers who helped
- [ ] Mantra ready: "I've prepared. I'm ready. I'll show what I know."
